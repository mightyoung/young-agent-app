> **引用说明**：
> - UI设计参考：[Material Design 3](https://m3.material.io/), [Apple Human Interface Guidelines](https://developer.apple.com/design/human-interface-guidelines/)
> - 动画设计参考：[Google Labs Flow](https://labs.google/flow/about)

### 1.1 产品定位
工业安全检查移动应用，为工业企业提供隐患上报、设备巡检、AI智能辅助等功能。

### 1.2 产品名称
- 中文名：Young-agent
- 英文名：MightYoung

### 1.3 目标用户
| 角色 | 权限范围 |
|-----|---------|
| 普通用户 | 隐患上报（全员开放） |
| 巡检人员 | 隐患上报 + 扫码检查 + 安全检查 + AI功能 |
| 管理人员 | 全部功能 + 设备管理 + 用户管理 + 报表查看 |

## 5. 离线策略

### 5.1 离线可用功能
| 功能 | 离线状态 |
|-----|---------|
| 扫码检查 | 可用，本地缓存检查项 |
| 隐患随手拍 | 可用，保存至本地 |
| 安全检查 | 可用，本地缓存任务 |
| 设备查看 | 可用，首次下载后缓存 |
| 消息中心 | 离线不推送，仅显示历史 |

### 5.2 数据同步策略
```
离线操作 → 本地SQLite存储 → 联网后同步
                    │
                    ▼
          ┌─────────────────┐
          │  冲突处理机制   │
          │  (用户手动选择) │
          │                 │
          │ [本地优先] [云端优先] │
          └─────────────────┘
```

### 5.3 消息推送
- 需要在线接收
- 发送失败则定期重试
- 失败消息本地记录

## 6. 数据模型

### 6.1 核心实体

| 实体 | 说明 | 核心字段 |
|-----|------|---------|
| User | 用户 | id, username, password, role_id, dept_id, enterprise_id |
| Enterprise | 企业 | id, name, logo |
| Department | 部门 | id, name, enterprise_id, parent_id |
| Device | 设备 | id, name, qr_code, type_id, location_id, dept_id, checklist_template_id |
| DeviceType | 设备类型 | id, name, code |
| DeviceLocation | 设备位置 | id, name, code, parent_id |
| ChecklistTemplate | 检查模板 | id, name, device_id, items (JSON) |
| InspectionRecord | 检查记录 | id, device_id, user_id, status, result, items, photos, created_at |
| HazardRecord | 隐患记录 | id, photos, type, location, description, status, user_id, created_at |
| Message | 消息 | id, type, title, content, user_id, read_status, created_at |

### 6.2 隐患类型 (单级分类)

| 类型编码 | 类型名称 |
|---------|---------|
| fire | 火灾安全隐患 |
| electric | 电力设施损坏 |
| construction | 违章建筑施工 |
| other | 其他环境安全 |

选择"其他"时支持用户自定义输入。

## 13. 动画效果设计

### 13.1 全局动画参数

| 参数 | 推荐值 | 说明 |
|-----|-------|------|
| 时长 | 200-300ms | 太快生硬，太慢拖沓 |
| 缓动 | ease-out | 进入缓出，退出缓入 |
| 延迟 | 0ms | 避免点击反馈延迟 |

### 13.2 页面切换动画

| 场景 | 动画类型 | 参数 | 参考 |
|-----|---------|------|------|
| 首页首次进入 | 浮现动画 | 800ms | 渐显效果 |
| 相机→扫码检查 | 滑动切换 | 300ms, ease-out | 水平过渡 |
| 相机→隐患随手拍 | 滑动切换 | 300ms, ease-out | 水平过渡 |
| 列表→详情 | 共享元素 | 300ms | 图片放大 |
| 底部弹起选择器 | 下推 | 250ms, ease-out | 从下往上 |
| 提交成功弹窗 | 淡入 | 200ms, ease-in-out | 透明度变化 |

### 13.3 微交互动画

| 场景 | 动画类型 | 参数 | 说明 |
|-----|---------|------|------|
| 按钮点击 | 缩放反馈 | scale: 0.95 | 点击时的按压效果 |
| 检查卡片切换 | 滑动选择 | 300ms | 参考探探(Tinder) |
| 语音输入 | 波纹动画 | 循环 | 参考微信 |
| Logo "Young-agent" | 流光/粒子围绕 | 持续 | 参考Google Labs Flow |
| 下拉刷新 | 弹性效果 | bounce | 惯性+回弹 |
| 列表滑动 | 惯性滚动 | spring | 物理惯性 |

### 13.4 动画性能优化

- 使用GPU加速：transform / opacity
- 避免触发重排：不在动画中修改layout
- 减少重绘：使用will-change提示浏览器
